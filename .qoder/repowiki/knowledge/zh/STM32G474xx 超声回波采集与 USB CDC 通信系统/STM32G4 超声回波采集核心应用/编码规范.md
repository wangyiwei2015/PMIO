- 所有用户扩展点统一包裹在 `/* USER CODE BEGIN x */` / `/* USER CODE END x */` 注释块内，以便 CubeMX 重新生成时不被覆盖。
- ISR 回调函数只做最小工作（读取寄存器、置 volatile 标志、计数），实际处理推迟到主循环完成，避免在中断上下文执行耗时操作。
- 外设初始化封装为 `static void MX_<Periph>_Init(void)` 风格函数，内部使用 `ADC_HandleTypeDef`/`GPIO_InitTypeDef` 结构体配置，错误路径统一调用 `Error_Handler()`。
- 共享状态变量（如 `trigger_detected`、`data_ready`、`uart_busy`）一律声明为 `volatile uint8_t`，并在访问前进行重入/边界保护（如 `remaining == 0`、`uart_busy` 屏蔽）。
