- 所有可被 CubeMX 重新生成的代码段都包裹在 `/* USER CODE BEGIN N */` / `/* USER CODE END N */` 注释块中，生成器只改写块外骨架。
- HAL PCD 回调函数使用 `#if (USE_HAL_PCD_REGISTER_CALLBACKS == 1U)` 条件编译，当启用动态回调注册时声明为 `static` 并通过 `HAL_PCD_RegisterCallback` 注册，否则作为全局符号由 HAL 直接调用。
- 每个 `USBD_LL_*` 底层接口统一采用 `HAL_StatusTypeDef hal_status = HAL_xxx(...); return USBD_Get_USB_Status(hal_status);` 的封装模式，将 HAL 状态映射到 `USBD_OK/FAIL/BUSY`。
- CDC 类通过全局 `USBD_HandleTypeDef hUsbDeviceFS` 和 `extern USBD_DescriptorsTypeDef CDC_Desc` 共享上下文，应用层与 CDC 适配层均通过该句柄间接访问类私有结构体。
