- 每个外设对应一对 `stm32g4xx_hal_<periph>.h` + `.c` 文件，并在同目录下提供可选的 `_ex.h/.c` 扩展接口。
- 对外 API 统一返回 `HAL_StatusTypeDef`，并通过 `__weak` 钩子（如 `HAL_MspInit`、`HAL_Delay`）允许用户覆盖默认实现。
- 使用 `__HAL_LINKDMA`、`__HAL_RESET_HANDLE_STATE`、`__HAL_LOCK/UNLOCK` 等 `__HAL_*` 宏集中管理 Handle 生命周期与 DMA 绑定。
- 所有可配置特性通过 `#if defined(...)` 或 `#if (X == 1U)` 编译期开关隔离，确保未启用外设的代码被完全剔除。
- 跨编译器差异通过 `cmsis_compiler.h` 及 `stm32g4xx_hal_def.h` 中的 `__weak/__packed/__RAM_FUNC/__ALIGNED` 等宏统一抽象，避免在业务代码中散写工具链特定属性。
